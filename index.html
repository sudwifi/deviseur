Comment ça fonctionne :
Le bouton "→ Axonaut" crée automatiquement un devis dans Axonaut avec toutes les lignes du chiffrage (matériel, MO, Adipsys). Le devis apparaît dans Axonaut, prêt pour relecture et envoi en signature électronique.

Alternative sans API : Si votre réseau bloque les appels directs, utilisez Make/N8N avec un webhook — je peux vous fournir le modèle.
